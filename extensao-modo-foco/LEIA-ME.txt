Modo Foco — extensão Chrome / Edge (colaboração com o app)
==========================================================

O que faz
---------
Quando o Pomodoro está em fase de foco com o timer rodando, o app liga automaticamente o
monitor de distrações e expõe o estado em http://127.0.0.1:<porta>/api/foco como JSON
(Authorization: Bearer <token>). Porta e token são gravados em bridge-config.json nesta pasta
quando o app inicia (127.0.0.1 apenas):
  focoAtivo (boolean), hosts (domínios), prefixosUrl (prefixos de endereço http/https),
  versaoBridge (número, uso interno).

A extensão compara o endereço completo da aba (URL sem fragmento #) com prefixosUrl; se bater,
substitui a página por "Site bloqueado". Se a API não mandar prefixosUrl (app antigo), cai no
match só por hostname em hosts.

Consultas ao app ocorrem: ao mudar URL (incluindo SPAs como Instagram, via API webNavigation),
ao concluir carregamento de aba, ao trocar de aba, e a cada 1 minuto (alarme) para revarrer
abas abertas. Após atualizar a extensão, use "Recarregar" em chrome://extensions ou edge://extensions.

Requisitos
----------
1. O aplicativo Modo Foco precisa estar ABERTO (o servidor local sobe com o app).
2. Carregar esta pasta como "extensão sem empacotamento" no Chrome ou Edge.

Instalação (Chrome)
--------------------
1. Abra chrome://extensions
2. Ative "Modo do desenvolvedor"
3. "Carregar sem compactação" → selecione a pasta extensao-modo-foco

Instalação (Edge)
------------------
1. Abra edge://extensions
2. Ative "Modo de desenvolvedor"
3. "Carregar sem compactação" → selecione a pasta extensao-modo-foco

Porta, token e lista de sites
------------------------------
A primeira porta tentada é 48721; se estiver ocupada, o app tenta 48722, … e atualiza
bridge-config.json. Abra o app Modo Foco antes da extensão usar o bloqueio; recarregue a
extensão se trocar de cópia da pasta ou reinstalar o app.

----------------------
Atualize a aba (F5) para voltar ao site normal.
